# RFM69

This is a port of the RFM69 library for arduino from https://github.com/LowPowerLab/RFM69 to python for raspberry pi.
Attach the RFM69 as follows:  

RFM pin - Pi pin  
3v3     - 17  
DIO0    - 18 (GPIO24)  
MOSI    - 19  
MISO    - 21  
CLK     - 23  
NSS     - 24  
Ground  - 25  
RESET   - 29
